# ChatGPT Memory Toolkit

> **專業級 Chrome 擴充套件** - 智能記憶管理、歷史追蹤、自動提醒，全方位 ChatGPT 記憶解決方案

## ✨ 核心功能

### 🔄 版本管理系統（NEW v1.1.0）
- **中央版本控制** - 統一管理所有組件版本，確保一致性
- **自動化更新** - npm scripts 自動同步 package.json 和 manifest.json
- **語義化版本** - 支援 patch、minor、major 版本遞增
- **版本驗證** - 內建版本一致性檢查機制

### 📚 歷史記錄管理（NEW v1.1.0）
- **完整追蹤** - 記錄所有匯出活動，包含時間戳、使用量、內容預覽
- **智能儲存** - 最大 50 筆記錄，自動清理舊資料防止溢出
- **視覺化瀏覽** - 優雅的歷史記錄介面，支援快速檢索和複製
- **一鍵操作** - 直接從歷史記錄複製內容到剪貼簿

### ⚙️ 應用設定頁面（NEW v1.1.0）
- **個人化設定** - 自動提醒開關、歷史記錄數量限制
- **智能清理** - 可配置的自動清理策略
- **設定持久化** - 使用 Chrome Storage API 安全儲存用戶偏好
- **即時生效** - 設定變更立即生效，無需重啟

### 🔔 自動提醒模態窗（NEW v1.1.0）
- **智能觸發** - 記憶使用量達 100% 時自動顯示優雅提醒
- **Material Design** - 現代化設計風格，非侵入式體驗
- **可控制性** - 用戶可透過設定頁面開啟/關閉自動提醒
- **一鍵行動** - 模態窗內直接執行匯出或稍後處理

### 📊 儲存使用量監控（NEW v1.1.0）
- **即時監控** - 動態顯示 Chrome Storage 使用量和配額
- **警告機制** - 使用量超過 80% 時提供清理建議
- **視覺化顯示** - 進度條和數值雙重展示，直觀易懂
- **優化建議** - 智能分析儲存模式，提供最佳化方案

### 🎯 自動檢測與匯出
- **智能監控** - 自動檢測「儲存的記憶已滿」提示
- **一鍵匯出** - 自動跳轉到記憶管理頁面並匯出
- **Markdown 格式** - 清晰易讀的 Markdown 格式輸出
- **剪貼簿複製** - 自動複製到剪貼簿，方便使用

### 🔍 精確檢測
- **記憶使用率檢測** - 支援中文介面的百分比檢測
- **記憶數量統計** - 準確計算記憶項目數量
- **即時狀態更新** - 動態顯示記憶狀態

### 🎨 現代化介面
- **Material Design** - 專業級使用者介面設計
- **狀態顯示** - 即時顯示記憶使用量和數量
- **響應式設計** - 適配各種螢幕尺寸和解析度
- **直觀操作** - 簡潔明瞭的操作流程

## 安裝方式

### 開發者模式安裝

1. **下載專案**

   ```bash
   git clone https://github.com/your-repo/chatgpt-memory-toolkit.git
   cd chatgpt-memory-toolkit
   ```

2. **版本初始化**（可選）

   ```bash
   # 安裝開發依賴
   npm install
   
   # 確保版本一致性
   npm run version:update
   ```

3. **載入擴充套件**
   - 開啟 Chrome，前往 `chrome://extensions/`
   - 開啟右上角的「開發人員模式」
   - 點擊「載入未封裝項目」
   - 選擇專案根目錄
   - 擴充套件安裝完成！

4. **驗證安裝**
   - 檢查工具列是否出現腦部圖標
   - 前往 [ChatGPT](https://chatgpt.com) 測試功能
   - 點擊擴充套件圖標檢查連接狀態

## 使用指南

### 🚀 自動模式（推薦）

1. **前往 ChatGPT** - 打開 [chatgpt.com](https://chatgpt.com)
2. **正常使用** - 當記憶接近滿載時會自動檢測
3. **智能提醒** - 記憶已滿時自動顯示優雅的提醒模態窗
4. **一鍵匯出** - 模態窗內直接執行匯出或選擇稍後處理
5. **自動儲存** - 匯出完成後自動儲存到歷史記錄

### 🎛 手動操作

1. **點擊圖標** - 點擊瀏覽器工具列的腦部圖標
2. **查看狀態** - 檢視記憶使用量、數量和儲存空間
3. **手動匯出** - 點擊「匯出記憶」按鈕
4. **複製資料** - 點擊「複製內容」按鈕
5. **查看歷史** - 點擊「歷史記錄」瀏覽過往匯出記錄
6. **調整設定** - 點擊「應用設定」自訂偏好選項

### 📚 歷史記錄功能

- **完整追蹤**: 所有匯出活動的時間戳和內容預覽
- **快速操作**: 直接從歷史記錄複製或刪除項目
- **智能管理**: 自動清理舊記錄，防止儲存空間溢出
- **視覺化瀏覽**: 優雅的介面設計，支援快速檢索

### ⚙️ 應用設定

- **自動提醒**: 控制記憶已滿時是否顯示模態窗
- **歷史數量**: 設定最大歷史記錄數量（10-100 筆）
- **自動清理**: 啟用/停用舊記錄自動清理功能
- **儲存監控**: 即時顯示 Chrome Storage 使用狀態

### 📄 輸出格式

匯出的 Markdown 格式包含：

- 記憶標題和使用量統計
- 記憶項目總數和匯出時間
- 完整的記憶清單（編號列表格式）
- 自動生成的元資料和分隔線

## 專業架構

### 📁 專案結構

```
chatgpt-memory-toolkit/
├── manifest.json          # Chrome 擴充套件配置
├── package.json           # 專案依賴和版本管理
├── src/                   # 核心程式碼
│   ├── scripts/           # 擴充套件腳本
│   │   └── content.js     # 內容腳本（主要邏輯）

├── assets/                # 靜態資源
│   └── icons/            # 圖標檔案
└── scripts/              # 自動化腳本
```

## ❓ 疑難排解

### 常見問題

**🔸 擴充套件圖標呈現灰色**

```
解決方案：
1. 確認在 ChatGPT 網站 (chatgpt.com)
2. 重新整理頁面
3. 檢查開發者控制台是否有錯誤
```

**🔸 無法檢測到記憶**

```
解決方案：
1. 前往 ChatGPT 的記憶管理頁面
2. 確保記憶功能已啟用
3. 嘗試重新載入擴充套件
```

**🔸 自動匯出未觸發**

```
解決方案：
1. 檢查是否出現「儲存的記憶已滿」文字
2. 手動點擊擴充套件圖標進行匯出
3. 查看瀏覽器控制台的錯誤訊息
```

### 偵錯模式

1. 開啟 `chrome://extensions/`
2. 找到 ChatGPT Memory Manager
3. 點擊「檢查檢視畫面」→「彈出式視窗」
4. 查看 Console 輸出
